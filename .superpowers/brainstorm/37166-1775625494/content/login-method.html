How should the admin log in?

Supabase supports several methods — pick what fits a single-admin clinic app.

A

Email + Password

Closest to what you have today. Admin enters email and password. Credentials managed in Supabase dashboard.

B

Magic Link (email OTP)

Admin enters email, receives a one-time login link. No password to remember or leak. Requires email access each login.

C

OAuth (Google / GitHub)

Admin logs in with an existing Google or GitHub account. Simplest for the admin — no new credentials needed.
